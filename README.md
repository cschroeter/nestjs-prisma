# primsa-nestjs

## Description

I've started working on an example project to integrate [Prisma](https://www.prisma.io/) in NestJS.
Supported features:

- [x] Project setup
- [x] Queries
- [x] Mutations
- [x] Subscriptions
- [x] Intellisense support

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```
